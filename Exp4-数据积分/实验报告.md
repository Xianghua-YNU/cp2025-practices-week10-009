## 二、实验原理

### 2.1 梯形积分法

对于离散的速度数据点$(t_0,v_0),(t_1,v_1),...,(t_N,v_N)$，总距离（位移）可通过梯形法则计算：

$$
\text{距离} \approx \sum_{i=0}^{N-1} \frac{v_i + v_{i+1}}{2} (t_{i+1} - t_i)
$$

### 2.2 累积距离计算

使用`scipy.integrate.cumulative_trapezoid`计算每个时间点的累积距离：

$$
\text{累积距离}(t_i) = \int_{t_0}^{t_i} v(t)dt
$$

// ... import numpy as np
from scipy.integrate import cumulative_trapezoid
import matplotlib.pyplot as plt
import os

def main():
    try:
        # 1. 获取数据文件路径（更智能的路径查找）
        current_dir = os.path.dirname(os.path.abspath(__file__))
        
        # 尝试可能的文件位置
        possible_paths = [
            os.path.join(current_dir, 'Velocities.txt'),  # 同级目录
            os.path.join(current_dir, '../Velocities.txt'),  # 上级目录
            os.path.join(current_dir, '../../Velocities.txt')  # 上上级目录
        ]
        
        data_file = None
        for path in possible_paths:
            if os.path.exists(path):
                data_file = path
                break
        
        if data_file is None:
            raise FileNotFoundError("未找到Velocities.txt文件")

        # 2. 读取数据
        data = np.loadtxt(data_file)
        t = data[:, 0]  # 时间列
        v = data[:, 1]  # 速度列

        # 3. 计算总距离（使用推荐的trapezoid替代trapz）
        total_distance = np.trapezoid(v, t)
        print(f"总运行距离: {total_distance:.2f} 米")

        # 4. 计算累积距离
        distance = cumulative_trapezoid(v, t, initial=0)

        # 5. 绘制图表（测试时不显示）
        if __name__ == '__main__':
            plt.figure(figsize=(10, 6))
            plt.plot(t, v, 'b-', label='速度 (m/s)')
            plt.plot(t, distance, 'r--', label='距离 (m)')
            plt.title('速度与距离随时间变化')
            plt.xlabel('时间 (秒)')
            plt.ylabel('速度 (米/秒) / 距离 (米)')
            plt.legend()
            plt.grid(True)
            plt.show()

    except FileNotFoundError:
        print(f"错误：找不到数据文件")
        print("请确保Velocities.txt文件位于以下位置之一：")
        print("\n".join(f"- {path}" for path in possible_paths))
    except Exception as e:
        print(f"发生错误: {str(e)}")

if __name__ == '__main__':
    main()

### 4.2 误差分析
分析梯形积分法计算距离的误差来源
1. 离散化误差（主要误差来源）
原因：
梯形积分法用直线连接相邻数据点，近似代替真实曲线，当函数非线性较强时，直线逼近会丢失曲率信息。

影响：

二阶误差(局部误差 ，全局误差 )

若速度变化剧烈（如急加速/减速），误差会显著增大。

2. 数据采样率不足
原因：
时间间隔 Δt 过大时，离散点无法捕捉速度的快速变化。

影响：

高频信号（如振动或噪声）可能被完全忽略。

根据 奈奎斯特采样定理，采样频率应至少为信号最高频率的2倍。

改进方法：

提高数据采样率（减小 Δt）。

对数据进行插值（如三次样条插值）后再积分。

3. 数据噪声
原因：
实验测量中的随机误差会导致速度数据波动。

影响：

噪声在积分过程中累积，导致距离计算漂移（随机误差→系统误差）。

高频噪声尤其影响结果（梯形法对高频分量敏感）。

改进方法：

滤波去噪（如移动平均、低通滤波）。

使用更鲁棒的积分方法（如 Simpson 法 或 高斯积分）。

4. 初始条件假设
原因：
梯形积分默认初始距离为0（initial=0），若实际初始位置非零，需额外修正。

影响：

系统误差（整体偏移）。

可通过校准初始位置消除。

5. 数值舍入误差
原因：
计算机浮点数运算的有限精度（尤其大量累加时）。

影响：

通常较小，但在极端情况下（如超长积分时间）可能显著。

可通过使用更高精度浮点（如 float64 而非 float32）缓解。
