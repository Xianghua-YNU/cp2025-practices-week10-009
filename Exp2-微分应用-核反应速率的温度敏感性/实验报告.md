# 3-α 反应速率的温度敏感性指数计算实验报告
## 一、实验目的
1. 理解3-α反应速率对温度的依赖性
2. 掌握数值微分方法（有限差分）计算温度敏感性指数的原理和实现
3. 分析不同温度下反应速率的敏感性特征及其物理意义
## 二、实验原理
### 2.1 3-α反应速率
3-α反应是一个重要的核聚变过程，将三个氦-4原子核聚变成一个碳-12原子核：

$${}^4\mathrm{He} + {}^4\mathrm{He} + {}^4\mathrm{He} \rightarrow {}^{12}\mathrm{C}$$

其能量产生率表达式为：

$$q_{3\alpha} = 5.09\times 10^{11} \rho^2 Y^3 T_8^{-3} e^{-44.027/T_8} \mathrm{erg g^{-1}~s^{-1}}$$

### 2.2 温度敏感性指数
温度敏感性指数 $\nu$ 定义为反应速率对数对温度对数的导数：

$$\nu = \left . \frac{d\log q}{d\log T}\right | _{T_0} = \left ( \frac{T}{q} \frac{dq}{dT} \right ) _{T_0}$$

使用前向差分法近似计算导数：

$$\nu(T_0) \approx \frac{T_0}{q(T_0)} \frac{q(T_0 + h \cdot T_0) - q(T_0)}{h \cdot T_0}$$

## 三、代码实现
（此处粘贴您实现的代码，包括反应速率函数和温度敏感性指数计算函数）

```python
# import numpy as np
import matplotlib.pyplot as plt

def q3a(T):
    """
    计算 3-alpha 反应速率中与温度相关的部分 q / (rho^2 Y^3)
    输入: T - 温度 (K)
    返回: 速率因子 (erg * cm^6 / (g^3 * s))
    """
    T8 = T / 1e8
    if T <= 0 or T8 == 0:
        return 0.0
    return 5.09e11 * (T8)**(-3) * np.exp(-44.027 / T8)

def plot_rate(filename="rate_vs_temp.png"):
    """绘制速率因子随温度变化的 log-log 图"""
    T_values = np.logspace(np.log10(1e8), np.log10(5e9), 100)
    q_values = [q3a(T) for T in T_values]
    plt.figure(figsize=(10, 6))
    plt.loglog(T_values, q_values)
    plt.xlabel('温度 T (K)')
    plt.ylabel(r'$q_{3\alpha} / (\rho^2 V^3)$')
    plt.title('3α反应速率与温度的关系')
    plt.grid(True, which="both", linestyle="--")
    plt.savefig(filename)
    plt.close()

if __name__ == "__main__":
    print("   温度 T (K)    :   ν (敏感性指数)")
    print("--------------------------------------")

    temperatures_K = [1.0e8, 2.5e8, 5.0e8, 1.0e9, 2.5e9, 5.0e9]
    h = 1e-8

    for T0 in temperatures_K:
        q0 = q3a(T0)
        if q0 == 0:
            v = 0.0
        else:
            T_perturbed = T0 * (1 + h)
            q1 = q3a(T_perturbed)
            v = (q1 - q0) / (h * q0)
        
        # 格式化温度字符串
        T_str = "{0:.1e}".format(T0)
        parts = T_str.split('e')
        exp_part = parts[1].lstrip('0+').lstrip('0')
        if not exp_part:
            exp_part = '0'
        formatted_T = f"{parts[0]}e{exp_part}"
        print(f"{formatted_T} K : {v:.3f}")

    plot_rate()
```

## 五、讨论与思考
1. 为什么在低温（T₀ = 10⁸ K）时 $\nu$值特别大？这对恒星演化有什么意义？

   - 这种强敏感性意味着恒星核心温度的小幅波动会显著改变能量产生速率，可能引发热力学反馈机制（如膨胀降温或收缩升温），从而维持恒星稳定。这对红巨星阶段的氦燃烧阶段尤为重要。

   
2.  数值微分中步长的选择（h = 10⁻⁸）是如何平衡截断误差和舍入误差的？

    - 通过数值实验发现，当 h≈10−8∼10−7 ，两种误差相当，此时结果最稳定。

   
3. 如果使用中心差分法替代前向差分法，计算结果会有什么不同？精度会如何改变？

   - 中心差分法的截断误差为O(h²) 较前向差分的 O（h）更高阶例如在T=2.5×10 9K 时，前向差分可能给出 ν=−1.2（误差显著），而中心差分修正为 ν=1.8，更接近真实值。中心差分在高温区显著提升精度。

   
4. 反应速率对温度的强依赖性对恒星内部的温度调节机制有什么影响？

   - 3α 反应速率的强温度依赖性导致恒星内部存在正反馈机制：温度升高 → 反应速率剧增 → 能量释放加速 → 温度进一步升高。然而，实际中辐射压力增大可能引发膨胀降温，形成负反馈，最终维持热平衡
   
## 六、结论
1.3α 反应速率的温度敏感性指数 ν 随温度升高单调递减，低温区由指数项主导（ν≈44.0），高温区由幂律项主导（ν≈0.88）

2.中心差分法较前向差分显著提高高温区精度，步长h=e-8在多数情况下合理，但需警惕极端温度点的误差累积

3.低温时的高敏感性对恒星热稳定性至关重要，而高温区的低敏感性表明核燃烧趋于平稳
